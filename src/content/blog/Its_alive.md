<h4 style="text-align: center"> The Start of a New Era </h4>

You may be wondering what prompted me to create a website for myself. This is a wonderful question. For a long time, I 
had been considering creating a new website to track all of my projects, so that I had a single location to provide
information and updates for all of my projects. Some might think that GitHub is a good source to point others to so that
they can be up-to-date. This is not a terrible option, except for the fact that my GitHub has become a mess. I plan to
one day clean up my GitHub so that others do not have to try and navigate old repositories, but that is not the focus of
this post. My true goal for this website is to have a one-stop shop for all of my major projects with commentary from me
on development details. This website will also serve as history for my projects, so that if future me has questions 
about what past me wrote I can just reference my previous details.


##### What are my plans for this website?
<details>
<summary></summary>
I do not have many plans as of now for my website. Currently, I aim to have blog posts for my projects. I also hope to
eventually leave GitHub behind and host the repositories myself. I do not plan on self-hosting my projects until I can 
host the server myself or can afford a server with a comfortable amount of storage. I potentially also plan on hosting
some sort of amateur radio over the internet system, such as EchoLink, but not certain on this.
</details>
